# System Addons

## Overview

This directory will soon host a variety of modules that can be added to enhance the functionality of the system. Currently, the modules directory is empty, but new modules will be introduced soon.

## Installing Addons

To install an addon, use the addon installer tool provided with the system. The addon installer ensures that the installation process is seamless and that all dependencies are properly managed.

### Steps to Install Addons

1. Open the addon installer.
2. Select the addon you want to install from the available list.
3. Follow the on-screen instructions to complete the installation.
